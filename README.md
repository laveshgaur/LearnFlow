LearnFlow is a **full-stack online education platform** built with **React, Node.js, Express, and MongoDB**.  
It allows students to browse courses, enroll, and track progress. Instructors can create and manage courses.

---

## Features

### Student
- Browse available courses
- Enroll in courses
- Track progress (lessons completed)

### Instructor
- Create, update, and delete courses
- Upload lessons (video/text placeholders)
- Manage enrolled students

### Common
- Authentication (Register/Login) with JWT
- Role-based access control (Student / Instructor)
- Responsive UI

---

## Tech Stack

- **Frontend:** React, React Router, Context API  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB (local or Atlas)  
- **Auth:** JWT, bcrypt  
- **Deployment:** Optional (Netlify / Vercel / Render)
